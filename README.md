# Rotten Potatoes

## Overview
Rotten Potatoes is a Ruby on Rails web application that allows users to browse, search, filter, and manage a database of movies.

## Key Features
- CRUD operations for movies
- Filtering and sorting by rating and release date
- Database-backed search functionality
- MVC architecture using Ruby on Rails

## Technologies
- Ruby on Rails
- ActiveRecord
- SQLite/PostgreSQL
- HTML/CSS
